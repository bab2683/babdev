# Stack

The Stack data structure follows the principle of last in, first out (LIFO) meaning that the last added item will be the first to be processed.

## Usages
- Functions invocations (call stack)
- undo/redo
- routing (back/forward)

## Big O

| Operation | Time |
| --------- | ---- |
| Insertion | O(1) |
| Deletion  | O(1) |
| Searching | O(n) |
| Access    | O(n) |
