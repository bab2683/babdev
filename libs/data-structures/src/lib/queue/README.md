# Queue

The Queue data structure follows the principle of first in, first out (FIFO) meaning that the first added item will be the first to be processed.

## Usages
- Video games queues
- Printer queues

## Big O

| Operation | Time |
| --------- | ---- |
| Insertion | O(1) |
| Deletion  | O(1) |
| Searching | O(n) |
| Access    | O(n) |
